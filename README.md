# 🎨 Classic Skin Morph

> ⚠️ **Current Status (PBE Only):** Currently, this tool is calibrated and tested exclusively on the **League of Legends PBE**. Once the classic assets and patches go live on the official retail servers, the launcher will automatically update to support them !

---

**Classic Skin Morph** is a free, open-source tool based on [LTK Manager](https://github.com/LeagueToolkit/ltk-manager) designed to clean up your game and restore the authentic, nostalgic feel of classic League of Legends. 

Let's be honest: seeing flashy, modern skins on a classic version of League is not only immersion-breaking, but **it's also bad for gameplay. You can easily get baited into thinking an opponent has their modern live-server kit instead of their classic one.** 

To solve this, **this tool forces the game to display only classic base models for all champions. If a specific classic model is not yet modeled or available, it cleanly defaults to the champion's standard base skin**, ensuring perfect visual clarity and a true throwback experience without game-breaking issues. **It also restores legacy splash arts and retro in-game HUD portraits !**

---

## ✨ Features

*   **Force Classic Skins:** Automatically overrides all skins to force the nostalgic, default base models. If a classic skin isn't modeled yet, it cleanly falls back to the standard default model.

*   **Champion Toggles:** Use the settings to easily disable the classic version of any specific champion and fall back to their modern default model.

*   **Legacy Visuals:** Restores original retro champion splash arts and classic in-game face icons (HUD portraits).

*   **Loading Screen & Borders:** A loading screen faithful to the Pre-Season 1 era, paired with a Season 3 ranked border system (with live ranks mapped to classic equivalents, from Bronze/Wood to Challenger). Fully togglable in the settings.

*   **Always Up-to-Date:** Our dedicated launcher automatically pulls the latest assets and updates them whenever Riot drops a patch.

*   **Open & Flexible:** Don't want to use the launcher? No problem. You can easily grab the raw `.fantome` files and use your favorite skin loader.
---

## 🚀 Installation & How to Use

We designed this tool to be as simple as possible. Just choose your preferred method:

### Option A: The Easy Way (Recommended)
Our lightweight launcher keeps your assets updated automatically and applies fixes as soon as League of Legends updates.

1.  Go to the [Releases](https://github.com/Xitfin/ClassicSkinMorph/releases) page.
2.  Download the latest version of the **Launcher** (`ClassicSkinMorph.exe`).
3.  Run the launcher, let it automatically sync the latest classic assets, and enjoy the classic Faille!

### Option B: Advanced / Manual Installation
If you prefer not to use our launcher, you can manually load the assets:

1.  Download the raw `.fantome` skin files directly from our release assets.
2.  Import them into **LCS Manager**, **Fantome**, or your preferred custom skin loader.
3.  Apply and play!
